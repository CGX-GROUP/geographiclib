00023    * \brief Properties of an ellipsoid
00024    *
00025    * This class returns various properties of the ellipsoid and converts
00026    * between various types of latitudes.  The latitude conversions are also
00027    * possible using the various projections supported by %GeographicLib; but
00028    * Ellipsoid provides more direct access (sometimes using private functions

00030    * Ellipsoid::InverseRectifyingLatitude, and Ellipsoid::MeridianDistance

00035    * should be limited to -3 < \e f < 3/4 (i.e., 1/4 < b/a < 4).
